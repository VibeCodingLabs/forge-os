# Antigravity IDE Profile

Antigravity is part of the default ForgeOS workstation profile while Forge Command Center is being built.

## Role

Use this lane for modern AI-assisted software development, project workspaces, and IDE-based development workflows.

## Install stance

Keep installation in a dedicated script because the official Linux packaging, login flow, and update method may change.

Reserved installer:

- `scripts/install-antigravity.sh`

## Workspace convention

Recommended workspace root:

- `~/forge-os/worktrees`

Recommended project root:

- `~/forge-os/projects`

## HP14 rule

Test this first on the HP14 lab profile before promoting it to the main workstation profile.
