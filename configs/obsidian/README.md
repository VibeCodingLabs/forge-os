# Obsidian AgenticOS Profile

Obsidian is the ForgeOS knowledge graph and operator notebook layer.

## Role

Use Obsidian for:

- project notes
- runbooks
- daily operator journal
- prompt libraries
- architecture maps
- task notes
- local markdown knowledge base
- plugin-driven dashboards

## Recommended vault layout

- `00-inbox/`
- `01-daily/`
- `02-projects/`
- `03-agents/`
- `04-runbooks/`
- `05-prompts/`
- `06-evals/`
- `07-telemetry/`
- `99-archive/`

## Install stance

Install method can be Flatpak, AppImage, deb package, or distro package depending on the machine profile.

Reserved installer:

- `scripts/install-obsidian.sh`

## HP14 rule

Test plugins and vault automation on the HP14 before using the same setup on the main workstation.
