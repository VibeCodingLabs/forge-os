# Model Event Fields

Each model interaction record should include:

- id
- created_at
- provider_id
- model_id
- agent_id
- task_id
- status
- input_tokens
- output_tokens
- total_tokens
- latency_total_ms
- tokens_per_second
- cache_hit
- context_injected
- retry_count
- selected_reason
- metadata_json
